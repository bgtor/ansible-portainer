v1.0.0
======

Release Summary
---------------

This collection contains a set of Ansible modules that allow management of Portainer environments and stacks. It is focused solely on Docker based environments. Currently Kubernetes environments are not supported.

New Modules
-----------

- portainer_config - Manage Portainer environment configs
- portainer_environment - Manage Portainer environments
- portainer_environment_info - Gets Portainer environment info
- portainer_group - Manage Portainer environment groups
- portainer_network - Manage Portainer environment networks
- portainer_secret - Manage Portainer environment secrets
- portainer_stack - Manage Portainer environment stacks
- portainer_tag - Manage Portainer tags
